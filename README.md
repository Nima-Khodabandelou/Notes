# java_core

Core concepts of Java SE

- Main Java features:

      Used for Desktop, Mobile, and Web application development
    
      Platform-independent
    
      Robust
    
      Secure
    
      High-level
    
      Multi-threaded
    
- Does not support:

      pointers (Handles it internally)
    
      goto
    
      operator overloading
    
      call by reference
    
      structures and unions
